This plugin provides custom functions to pull Messari data directly into your sheet.

Returns the most recent timeseries value for a metric.

=MESSARI_LATEST_POINT("asset-price", "open", "bitcoin")

Returns a single timeseries value at a specific date or timestamp.

=MESSARI_POINT("asset-price", "close", "ethereum", "2024-01-01")

Returns a 2D array of historical values between a start and end date, perfect for creating charts.

=MESSARI_SERIES("asset-price", "high", "bitcoin", "2024-01-01", "2024-01-31")
